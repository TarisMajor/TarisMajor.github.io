GestureAI Demo

GestureAI is an app that uses machine learning to recognize and interpret hand gestures. It was created using Python, OpenCV, and TensorFlow.

Scheduler Demo

Developed a CPU scheduling simulator in Python that models and compares four common algorithms: First-Come-First-Serve (FCFS), Priority-Based (PB), Round Robin (RR), and Multi-Level Feedback Queue (MLFQ). The program visualizes process execution over time and outputs performance metrics such as average wait time and turnaround time for each algorithm, helping analyze their efficiency under different workloads.

Shell Demo

Built a custom Unix-like shell in Python that emulates core functionalities of a WSL Ubuntu environment. Supported commands include ls, grep, cd, history, and more , with features such as command parsing, error handling, and persistent history. This project demonstrates an understanding of shell behavior, subprocess management, and user input parsing in Python.

A Tranter Heat MaxChanger Fabricated using Additive Manufacturing

This project was part of a team of Mechanical Engineering students at Midwestern State University. We redesigned and fabricated a heat maxchanger using 3D printing and additive manufacturing.

A Continuously Variable Gear Ratio Multistage Gearbox for Wind Turbines

This project simulates a variable gear ratio gearbox for wind turbines using Matlab . The gearbox is designed to be used in wind turbines to improve efficiency and reliability while removing the need for power electronics.
